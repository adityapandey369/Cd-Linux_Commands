# Cd-Linux_Commands
This is a simple repository which tries to note down some of the important and basic linux command for different use cases

1. Extract .tar file : tar -xf "file name"
2. Make installation file executable : chmod +x ros_humble.sh
                                       ./ros_humble.sh
3. See the connected usb : ls /dev/ttyUSB* or dmesg -w
4. Provide permission to your user : usermod -aG dialout $USER or sudo chmod 777 /dev/ttyUSB0
